# About

This is the personal blog for `woncomp`, built with `Jekyll` and the `Chirpy` theme.

The site is set up for Markdown-first writing and GitHub Pages deployment. It is intentionally lightweight so new posts, notes, and longer technical articles can be published without extra tooling.

## What lives here

- Blog posts written in Markdown
- Notes, experiments, and references worth keeping
- Longer-form technical writing as the site evolves

## Status

This is the first site setup. Branding, favicon, social links, comments, and custom domain settings can be added later without changing the writing workflow.
