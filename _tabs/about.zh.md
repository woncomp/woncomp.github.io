# 关于

这是 `woncomp` 的个人博客，使用 `Jekyll` 构建，采用 `Chirpy` 主题。

本站专注于 Markdown 写作和 GitHub Pages 部署。采用轻量化设计，无需额外工具即可发布文章、笔记和长篇技术内容。

## 内容

- Markdown 格式的博客文章
- 值得保留的笔记、实验和参考资料
- 随着站点发展不断丰富的长文技术写作

## 状态

这是站点的初始搭建。品牌标识、头像、社交链接、评论系统和自定义域名等设置可以在后续逐步添加，不会影响当前的写作流程。
